#Syntecxhub_Project_BankingManagementSystem

#Overview
This project is developed as part of my Syntecxhub Java Internship (Week 3 Task).

It is a console-based Banking Management System built using Core Java,
which allows user to manage customer bank accounts easily. The system supports creating accounts,
displaying all accounts, searching, updating, deleting, and performing transactions such as
deposit and withdrawal. It also includes file handling so that account records are saved permanently.

#Features
✅ Create new bank account
✅ Display all bank accounts
✅ Search account by Account Number
✅ Update customer details
✅ Delete bank account
✅ Deposit money
✅ Withdraw money (with balance validation)
✅ Prevent duplicate Account Numbers
✅ Validate transaction amounts
✅ Fully modular OOP-based system

#Technologies Used
Java (OOP + Collections)

ArrayList for data storage

Scanner for input handling

Any Java-supported IDE (VS Code / IntelliJ / Eclipse)

GitHub for version control

#Project Structure
src/
   ├── Account.java // Account model class
   ├── BankManager.java // Handles all operations (CRUD + transactions)
   └── BankingManagementSystem.java // Main program menu
